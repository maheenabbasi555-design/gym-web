# Gym Management System

## Overview
This is a **Gym Management System website** built using **Flask, HTML, and CSS**.  
The website allows users to explore gym details, learn about the team, and view membership plans.

## Pages
- **Home:** Displays a welcome message and gym highlights.
- **About Us:** Shares details about the gym's mission, team, and facilities.
- **Membership:** Showcases membership plans and services.

## Project Structure
# gym-web
A Flask-based Gym Management System website with Home, About, and Membership pages.
